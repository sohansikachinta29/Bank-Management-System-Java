# Bank Management System (Java)
This is a console-based Bank Management System developed using Java and Object-Oriented Programming concepts.

## Features
- Create Savings or Current account
- Deposit money
- Withdraw money
- Check balance
- Calculate interest
- Display account details

## OOP Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation

## How to Run
1. Compile the program:
   javac Bank.java
2. Run the program:
   java Bank
